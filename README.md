# Celestia

> 二游集成启动器

## 核心功能

* **游戏集成** ：支持添加本地已安装的二次元游戏，配置游戏名称、可执行文件路径及封面图
* **一键启动** ：绕过各厂商的官方启动器，直接调用游戏本体 exe 启动游戏
* **游戏管理** ：统一管理所有已添加的游戏，支持编辑和删除

## 计划功能

* 游戏时长记录
* 账号数据看板（游戏内数据展示）
* 游戏更新提醒

## 技术栈

* **框架** ：Electron
* **语言** ：TypeScript
* **UI** ：React
* **构建** ：Webpack

## 开发环境

* Node.js v22 LTS
* npm

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/jenson2525/celestia.git
cd celestia

# 安装依赖
npm install

# 启动应用
npm start
```
